# Cypress Test

## SetUp

1. Configure a `cypress.json` file based on [cypress.sample.json](./cypress.sample.json) 

1. Run
```
npm install
```

## Running


#### Using cypress UI mode
```bash
npx cypress open
```
#### Using Headless mode
```bash
npx cypress run
```
#### Run specific test in Headless mode
```bash
npx cupress run --testFile.js
```
